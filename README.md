# ichux-lista-10
-Equipe: Cauã Furst, João Magalhães, Luiz Guilherme
-Criamos um aplicativo semelhante ao ifood para otimizar o tempo de espera na fila.
Tela 1 – Login
Campos para usuário e senha
Botão de login
Interface simples para autenticação do usuário
Tela 2 – Busca de estabelecimentos
Campo de pesquisa para locais próximos
Lista de opções (ex: McDonald's, Burger King, Bob’s)
Seleção de um estabelecimento
Tela 3 – Escolha de item/serviço
Opção de selecionar tipo de item (ex: modelos de hambúrguer)
Exibição visual com imagens
Destaque para seleção do usuário
Tela 4 – Confirmação de escolha
Alternância/opção adicional (ex: habilitar localização)
Confirmação do item escolhido
Pop-up para confirmar ou cancelar ação
Tela 5 – Resumo do pedido
Exibição do pedido realizado
Tempo estimado de preparo/entrega
Lugar na fila
Opção de avaliação (estrelas)

-O fluxo principal é a sequência de ações essenciais que o usuário realiza para completar a função principal ao sistema.
